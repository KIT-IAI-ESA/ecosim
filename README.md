# eCoSim

A python-based distributed co-simulation software for multi-physics models

## About

eCoSim is an open-source, multi-domain co-simulation software for sector-coupled energy system analysis. It is part of the **eASiMOV** software suite.

Check the full suite on our page: **[easimov.com](https://easimov.com)**
